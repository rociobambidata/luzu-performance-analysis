# Luzu TV Performance Analysis

## Descripción
Análisis exploratorio de la performance general de Luzu TV utilizando Python, Google Colab y visualización de datos.

## Objetivo
Evaluar métricas clave de rendimiento, identificar patrones de audiencia y generar insights útiles para la toma de decisiones.

## Herramientas
- Python
- Pandas
- Matplotlib & Seaborn
- Google Colab
- Presentación final (PDF/PPT)

## Contenido
- Limpieza y preparación de datos
- KPIs de views, alcance y engagement
- Comparación entre programas y formatos
- Insights accionables

## Archivos
- `Luzu TV - Análisis de performance.ipynb` — Notebook con el análisis completo
- `Luzu - Análisis de performance.pdf` — Presentación final

## Principales Insights
- Tendencias de engagement por programa
- Variables que más afectan el alcance
- Evolución temporal del rendimiento
